XML is a very important markup language for data storage across the internet and it has the following purposes :

XML supports the structuring of data for its easy storage and transport . XML tags give structure to the data .
XML eases data sharing .
XML stores data in text format plain text on the databases .
XML makes data display easier for HTML . If the data is dynamic it is difficult to edit the HTML document each time before displaying it . Using XML the data gets stored in independent XML files and any changes to this data will not need any change in HTML .

The major difference between XML and HTML markup languages is that HTML is used for displaying data on the website whereas XML is used for storing and structuring the data in the database .
